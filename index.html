AMIE : Application Mobile pour les Intervenants de l'Environnement

Ce projet a été réalisé par des étudiants de l'ENSG, l'École Nationale des Sciences Géographiques. Ce projet s'est réalisé à la demande de l'OFB, l'Office Français de la Biodiversité. Il s'agissait de créer une application mobile accessible hors ligne pour visualiser les données d'observations à des fins d'utilisation sur le terrain. Aussi pour créer des jeux de données ergonomiques, nous avons créé un outil Tuilator permettant de générer des jeux de données sous forme de tuiles vectorielles et raster. Voici notre projet AMIE, Application Mobile pour les Intervenants de l'Environnement, composé de son application mobile BioViz et de l'outil de tuilage SmartTuileur.

BioViz : notre application mobile

L'outil central de ce projet est l'application mobile à destination des agents de l'OFB.

Visualiser

Il s'agit de proposer une visualisation ergonomique des données d'observation déjà existantes

Hors ligne

L'idée c'est d'y accéder directement sur le terrain, où le réseau n'es pas toujours présent.

Localiser

L'agent doit pouvoir accéder à sa position pour analyser son environnement à l'aide des données d'observation.

Mesurer

Proposer un outil afin de mesurer une longueur ou une surface donnée sur le terrain.

SmartTuileur : Générateur de jeux de données tuilés

L'OFB dispose de nombreux jeux de données.
